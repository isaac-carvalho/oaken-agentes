# Sobre o portfólio Oaken Agentes

Este portfólio contém 21 projetos práticos cobrindo as carreiras Alura de Especialista em IA e Engenharia de Agentes de IA.
Cada projeto roda com um único comando e é estruturado para ser auditável por recrutadores em até 3 minutos.
A arquitetura comum reaproveita um cliente LLM unificado em `projects/_shared/`, suportando OpenAI, Gemini e Anthropic com fallback mock para execução offline.
